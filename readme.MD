```
- Git
  - init  : tạo repo git trong local
  - clone : clone repo từ github
  - add   :
    + tên file : add duy nhất file đó
    + . add tất cả trong repo local những file chưa add
  - commit: commit những file vừa add
  - status: check status
  - push  : đồng bộ repo local với github
  - pull  : đồng bộ repo từ github với local
  - log   : xem log
  - brand : Quản lý branch
  - merge : hợp nhất các brand vào nhánh chính
  - remote:
    + add : add remote
    + remove: remove remote
  - checkout: switch sang brand khác
```
